### Installation

Clone this repository:

```bash
git clone https://github.com/Yumna0019/Quiz-App.git
```

Navigate to the project directory:

```bash
cd Quiz_App
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
